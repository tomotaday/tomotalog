# ともたろぐ

生活を振り返るためのカレンダー型家計簿

**関数よりPivotが好きだった。**

---

# コンセプト

ともたろぐは、家計簿を作るために生まれたアプリではありません。

一つの生活データを、さまざまな切り口から何度でも振り返るための「生活ログ」です。

Googleマップのタイムラインが「時間」で生活を振り返るように、
ExcelのPivotTableが「視点」を変えてデータを眺められるように、

ともたろぐは、その二つを融合したアプリを目指しています。

**データは一つ。見方は無限。**

年月が経つほど価値が増していく、
「データの複利」を実現することが、ともたろぐ Version2 の設計思想です。

Version2では、

「家計簿を見る」のではなく、

**「暮らしを振り返る」**

そんな体験を目指しています。

Ver2.4では、

GoogleタイムラインJSONを実際に取り込みながら、

場所図鑑・生活ログ・インポート基盤を
実データで育てる研究を開始しました。

データを書き換えるのではなく、

**その時点で有効だった事実を積み重ねる。**

これが、ともたろぐ Version2 の中心思想です。
