# ともたろぐ

生活を振り返るためのカレンダー型生活ログ

**関数よりPivotが好きだった。**

---

# コンセプト

ともたろぐは、家計簿を作るために生まれたアプリではありません。

一つの生活データを、
さまざまな視点から何度でも振り返るための
**生活ログ**です。

Googleマップのタイムラインが
「時間」で生活を振り返るように、

ExcelのPivotTableが
「視点」を自由に切り替えられるように、

ともたろぐは、その二つを融合したアプリを目指しています。

**データは一つ。見方は無限。**

年月が経つほど価値が増していく

**「データの複利」**

これが Version2 の設計思想です。

# Version2 の考え方

Version2では、

生活データを役割ごとに分離して管理します。

```text
Googleタイムライン研究所
（隔離実験区域）
        │
        │ データ共有なし
        ▼

正式タイムライン格納庫
（生活履歴）

        │
        ▼

訪問図鑑
（場所マスター）
```

研究所は、

UIや処理方法を検証するための実験場です。

正式実装では、

タイムライン格納庫が生活履歴を管理し、

訪問図鑑が場所そのものを育てます。
